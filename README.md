# Mini Games Hub

A growing collection of simple, self-contained mini games — built for fun and practice. Each game lives in its own folder with its own code and its own README explaining how it works.

---

## Structure

This repo is organized as one folder per game:

```
mini-games-hub/
├── TicTacToe/
├── RockPaperScissors/
├── MemoryCard/
├── GuessTheNumber/
└── README.md
```

Every game folder contains its own source code and a `README.md` with details, rules, and how to run it.

## Getting Started

Clone the repo:

```bash
git clone https://github.com/FaiqaAamer/Mini-Games-hubs.git
cd Mini Games
```

Then go into any game folder and follow its own README for setup/run instructions.

---

## Contributing

Want to add a new mini game?

1. Create a new folder for it.
2. Add your code + a `README.md` inside that folder.
3. Add a link to it in the **Games** section above.
4. Open a pull request.
